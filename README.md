# LayerSpy ⚡
> A modern, client-side, browser-based 3D Print G-Code Analyzer and Live-Tuner.

**Website:** [https://layerspy.de](https://layerspy.de)

## What is LayerSpy?
LayerSpy is a privacy-first web application designed for 3D printing enthusiasts. It allows you to analyze, tune, and simulate your G-Code files completely locally inside your browser—without uploading proprietary 3D models to any cloud server.

### 🌟 Key Features
- **100% Local Processing:** Your G-Code never leaves your computer.
- **2D & 3D Visualization:** Explore toolpaths interactively using WebGL (Three.js).
- **Speed Heatmaps:** Identify bottlenecks and cooling issues visually.
- **Live Parameter Tuning:** Adjust hotend/bed temperatures, fan speeds, flow, and print speeds dynamically.
- **Klipper & Marlin Kinematics:** Simulate realistic print times using advanced kinematics math (Jerk, Acceleration, Max Velocity).

## Why build this?
Many users rely on desktop slicers (like PrusaSlicer, SuperSlicer, or Cura) just to quickly check a G-Code file. LayerSpy provides a lightweight, instant alternative directly in the browser.
